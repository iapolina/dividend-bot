# 📊 Дивидендный бот 

Автоматический парсер дивидендных акций Московской биржи.  
Собирает данные, рассчитывает доходность, фильтрует акции с доходностью > 10% и записывает результаты в Google Таблицу.

---

## ✨ Возможности

- Парсинг цен и дивидендов с Мосбиржи и dohod.ru
- Отбор акций с дивидендной доходностью выше банковского депозита (по умолчанию 10%)
- Расчёт:
  - Доходности выплаты
  - Суммы дивидендов на один лот
  - Количества лотов для получения 1000 ₽ дивидендов
  - Стоимости покупки для 1000 ₽ дивидендов
- Запись данных в Google Таблицу с автоматическим обновлением
- Дата последнего обновления под таблицей
- Чёрный список тикеров (для исключения мусорных бумаг)
- Ручные данные для тикеров с низкой ликвидностью (например, URSB)

## 🚀 Быстрый старт

1. Клонируй репозиторий

```bash
git clone https://github.com/iapolina/dividend-bot.git
cd dividend-bot

2. Установи зависимости
bash
pip install -r requirements.txt


3. Настрой Google Таблицу
3.1. Создай сервисный аккаунт в Google Cloud
Перейди в Google Cloud Console

Создай проект (или выбери существующий)

Включи Google Sheets API и Google Drive API

Создай сервисный аккаунт и скачай JSON-ключ → переименуй в credentials.json

3.2. Создай Google Таблицу
Создай новую таблицу и назови её, например, «Дивидендные НЕсушки»

Скопируй ID таблицы из URL:
https://docs.google.com/spreadsheets/d/ЭТО_ТВОЙ_ID/edit

3.3. Дай доступ сервисному аккаунту
Скопируй email из credentials.json (поле client_email)

Нажми «Поделиться» в таблице → добавь этот email с правами «Редактор»

4. Настрой переменные окружения
Создай файл .env в корне проекта:

env
SPREADSHEET_ID=твой_id_таблицы
CREDENTIALS_FILE=credentials.json

5. Запусти скрипт
bash
python main.py

⚙️ Управление списком акций
Чёрный список
В файле main.py есть список BLACKLIST — туда можно добавить тикеры, которые бот должен игнорировать (например, акции без дивидендов или с нулевой ликвидностью).

Ручные данные
Для тикеров, которые не парсятся автоматически (например, низколиквидные), предусмотрен словарь MANUAL_DATA. Ты можешь добавить в него цену, лотность и дивиденды вручную.

⚠️ Важно
Никогда не заливай файлы credentials.json и .env в репозиторий — они содержат секретные ключи!

Скрипт перезаписывает данные в таблице, поэтому не редактируй вручную колонки, которые он заполняет (лучше добавь свои колонки справа от них)

Сайт dohod.ru может блокировать частые запросы — добавлена задержка между запросами
